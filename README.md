python program that checks your current theme via darkdetect  
REQUIRES:  
python 3  
darkdetect  
, for linux, you may need a venv  
no warranties!  
if you see an issue, instead of making an issue, fork and pull request  

to install dependencies:  

```bash

python -m venv path/to/dir

pip install darkdetect

```
and run the program via `python -m current-theme.py`
